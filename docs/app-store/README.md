# App Store Submission Assets

This folder contains all assets needed for App Store submission.

## Files

- **description.md** - Full App Store listing copy (name, subtitle, description, keywords)
- **keywords.txt** - App Store search keywords (100 char limit)
- **screenshots/** - App Store screenshot images (see requirements below)

## Screenshot Requirements

### iPhone 6.5" Display (Required)
- **Resolution:** 1284 x 2778 pixels
- **Format:** PNG or JPEG
- **Quantity:** 3-10 screenshots (5-6 recommended)
- **Device:** iPhone 14 Pro Max, iPhone 13 Pro Max, iPhone 12 Pro Max

### Recommended Screenshots

1. **Quote Creation** - Show the main quote creation screen
2. **Material Picker** - Display the product catalog and selection
3. **Professional PDF** - Show generated quote PDF preview
4. **Dashboard** - Display quote list/management
5. **Assembly Templates** - Show pre-built job templates
6. **Settings/Branding** - Show company customization options

### Screenshot Tips

- Add marketing text overlays highlighting key features
- Use consistent color scheme (orange #f97316 accent)
- Show real-world use cases
- Include device frames for polish
- Test on actual device sizes

### Tools for Screenshots

- **Simulator** - Use Xcode simulator to capture at correct resolution
- **Real Device** - Capture on iPhone 14 Pro Max
- **Design Tools:**
  - Figma (free, recommended)
  - Canva (for text overlays)
  - Rotato (for device frames)

## App Store Connect Info

**Category:** Business > Productivity
**Age Rating:** 4+
**Content Rights:** You own all rights to the content

## Next Steps After Apple Approval

1. Generate app screenshots using simulator
2. Upload screenshots to `screenshots/` folder
3. Review and finalize `description.md` content
4. Set up App Store Connect listing
5. Upload build via Xcode or Transporter
6. Submit for review
