## Impact on Contractors

These gaps are **critical for contractors** because:

1. **Framing studs** - The most basic material for any construction project. A contractor cannot frame a wall without untreated 2x4 studs.

2. **Plywood/OSB** - Essential for:
   - Subfloors
   - Wall sheathing
   - Roof decking
   - Cabinet backing

3. **Drywall panels** - Required for every interior finishing job. Having only drywall accessories without actual sheets is useless.

Without these materials, **Menards cannot be used as a primary supplier** in QuoteCat, which defeats the purpose of having local pricing data. Contractors will see Menards prices but won't be able to quote the most common materials.

## Verification Queries Used

```sql
-- Plywood/OSB counts
SELECT supplier_id, COUNT(*) FROM products
WHERE LOWER(name) LIKE '%plywood%' OR LOWER(name) LIKE '%osb%'
GROUP BY supplier_id;

-- Untreated 2x4 studs
SELECT supplier_id, name FROM products
WHERE supplier_id = 'menards'
AND (LOWER(name) LIKE '%2 x 4%' OR LOWER(name) LIKE '%2x4%')
ORDER BY name;

-- Drywall panels vs accessories
SELECT supplier_id, name FROM products
WHERE supplier_id = 'menards'
AND LOWER(name) LIKE '%drywall%'
ORDER BY name;
```

**Menards data is missing the most fundamental building materials:**
