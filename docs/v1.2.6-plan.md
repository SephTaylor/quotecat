## What v1.2.6 delivers

**Pricing Health Check** — a Pro-only screen reachable from the Toolbox under Pro Tools. The contractor taps it, the screen analyzes their last 90 days of sent/approved/completed quotes, and surfaces ones where the actual profit margin was meaningfully below their target. Shows estimated lost profit per quote and across the portfolio.

The math is the same `calculateQuoteProfitability` function powering the per-quote margin indicator that became Free in v1.2.5 — so any quote that showed a red indicator on its edit screen will surface in this audit. Consistency between the in-the-moment view and the historical view.

## Analysis rules

For each quote with `status IN ('sent', 'approved', 'completed')` from the last 90 days:

1. **Compute profitability** via `calculateQuoteProfitability`. Returns `null` if labor rates aren't set — those quotes are silently skipped (counted in `totalSkipped` and surfaced as a banner if the count is high).
2. **Resolve target margin:** prefer `userPrefs.overhead.targetProfitMarginPercent`; fall back to default `20%` if unset (and surface a "using default" banner).
3. **Flag if** `actualMargin < targetMargin - 5`. This matches the existing red-zone threshold from `getMarginColor` so the audit verdict and the live indicator stay consistent.
4. **Estimated lost profit** = `((targetMargin - actualMargin) / 100) * revenue`. This is the **margin shortfall at the price actually sold**, NOT what would be made at a reprice. Surfaced honestly via the info-tap tooltip on every dollar number.
5. **Sort by lost profit descending,** return the top 10.

## Resolved decisions (already implemented)

1. **Threshold:** `actualMargin < targetMargin - 5`. Matches the live margin indicator's red zone.
2. **Time window:** 90 days fixed. No toggle in v1.
3. **Tier gating:** Pro-only. RevenueCat paywall via `presentPaywallAndSync()`. On cancel, `router.back()`.
4. **Labor cost handling:** Skip un-analyzable quotes silently. Surface "set your labor rates" banner if skipped count is high.
5. **Lost-profit tooltip:** Info icon on hero number + every row. Tap opens explainer alert with: *"Margin shortfall — the profit gap between your target margin and what this quote delivered, at the price you actually sold. Not what you'd make if you repriced."*
