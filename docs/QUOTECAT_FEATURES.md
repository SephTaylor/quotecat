# QuoteCat - Complete Feature Guide

QuoteCat is a construction quoting platform designed to help contractors create professional quotes, invoices, and contracts in minutes instead of hours. It consists of a mobile app (iOS) and a web portal for client interactions.

---

## Who Is QuoteCat For?

- **Contractors** (electricians, plumbers, HVAC, general contractors)
- **Small trade businesses** (1-20 employees)
- **Field service professionals** who need to quote jobs on-site

---

## Mobile App Features

### Dashboard
Your business at a glance:
- **Quick stats** - See counts for drafts, sent quotes, approved quotes, and items needing invoicing
- **Business value tracking** - Total value of sent quotes, approved quotes, and pending invoices
- **Recent activity** - Quick access to recent quotes, invoices, and contracts
- **Cloud sync status** - See when your data was last backed up (Pro/Premium)

### Quotes

**Creating Quotes**
- Add project name and client details (name, email, phone, address)
- Add materials from the product catalog or manually
- Set labor costs (hours and rate)
- Configure markup percentage
- Add tax percentage
- Write notes visible to client
- Set follow-up reminders

**Quote Organization**
- **Status workflow**: Draft → Sent → Approved → Completed → Archived
- **Search and filter** by status, client name, or project name
- **Pin important quotes** to keep them at the top
- **Bulk operations** - Select multiple quotes to change status, archive, or delete

**Multi-Tier Pricing (Bundles)**
- Create "Good/Better/Best" pricing options from any quote
- Quotes are linked together in a bundle
- Client sees all options side-by-side
- When one option is accepted, others are automatically archived
- Export all options as a single PDF

**Quote Actions**
- Generate professional PDF
- Share via email, text, or any app
- Duplicate quote as template
- Convert to invoice (Pro+)
- Convert to contract (Premium)

### Invoices (Pro/Premium)

**Invoice Management**
- Create invoices from approved quotes
- Automatic invoice numbering
- Set due dates
- Track payment status (Unpaid, Partial, Paid, Overdue)
- Record partial payments

**Invoice Actions**
- Generate PDF
- Send to client via email
- Client pays online via Stripe
- Mark as paid manually

### Contracts (Premium)

**Digital Contracts**
- Create contracts from approved quotes
- Professional contract formatting
- Send to client for e-signature
- Track when client views and signs
- Legally binding digital signatures

### Clients (Pro/Premium)

**Client Manager**
- Save client contact information
- Reuse clients across quotes and invoices
- View client history (all quotes, invoices, contracts)
- Quick client selection when creating quotes

### Assemblies (Pro/Premium)

**What Are Assemblies?**
Assemblies are templates for common jobs. Instead of adding items one-by-one, select an assembly and all the materials are added automatically.

**Assembly Library**
- Pre-built assemblies for common tasks
- Browse by category (electrical, plumbing, HVAC, etc.)
- See materials and pricing before adding

**Assembly Manager**
- Create your own custom assemblies
- Set dynamic quantities (e.g., "outlets per room")
- Include labor estimates
- Reuse across all your quotes

### Drew - AI Quote Wizard (Premium)

**What Is Drew?**
Drew is an AI assistant that helps you build quotes through conversation.

**How It Works**
1. Tell Drew about the job ("I need to wire a 2000 sq ft house")
2. Drew generates a material checklist based on trade knowledge
3. Review and adjust quantities
4. Drew calculates labor and creates your quote
5. Save and send to client

**Drew Features**
- Natural language understanding
- Trade-specific knowledge (electrical, plumbing, HVAC, etc.)
- Voice input support
- Product suggestions from your pricebook
- Labor estimation

### Pricebook (Pro)

**Custom Product Catalog**
- Create your own products with custom pricing
- Import products in bulk
- Organize by category
- Use your products in quotes and assemblies

### Settings

**Company Details**
- Company name and logo
- Contact information (email, phone, website)
- Business address
- Logo appears on all PDFs

**Business Defaults**
- Default markup percentage
- Default labor rate
- Default tax rate

**Appearance**
- Light and dark themes

**Account**
- Sign in with email or Apple ID
- Biometric login (Face ID / Touch ID)
- View subscription status

### Cloud Sync (Pro/Premium)

**Automatic Backup**
- All data synced to the cloud
- Access from multiple devices
- Never lose your quotes
- Pull-to-refresh to sync manually

- Professional quote page with your branding

**Signing Contracts**
