## What is QuoteCat?

QuoteCat helps contractors and builders create professional construction quotes in minutes. Select materials from our catalog, add labor costs, and generate PDF quotes on-site.

**Key Features to Try:**
- Create quotes with 100+ construction products
- Custom assemblies (groups of materials)
- PDF & CSV export with company branding
- Dashboard with quote value tracking
- Light/dark mode with beautiful gradients

## What We Need From You

**Your Mission:**
Try to create a real quote for a project (even a small one). Walk through the entire workflow as if you were quoting a job.

**Test These Scenarios:**
1. Create a quote from scratch
2. Add 5-10 products from the catalog
3. Add labor costs
4. Generate a PDF - does it look professional?
5. Edit an existing quote
6. Try the Assembly Manager (create a template)

**Look for:**
- Confusing navigation
- Bugs or crashes
- Missing features
- Slow performance
- Layout issues
