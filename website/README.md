## How to Deploy to GoDaddy

### Option 1: File Manager (Easiest)

1. Log into your GoDaddy account
2. Go to **Web Hosting** â†’ **Manage**
3. Click **File Manager**
4. Navigate to `public_html` folder (or your domain's root folder)
5. Delete any existing `index.html` if present
6. Click **Upload** button
7. Upload the `index.html` file from this folder
8. Done! Visit https://quotecat.ai to see it live

### Option 2: FTP (If you prefer)

1. Get your FTP credentials from GoDaddy (cPanel â†’ FTP Accounts)
2. Use FileZilla or any FTP client
3. Upload `index.html` to the `public_html` folder
4. Done!
